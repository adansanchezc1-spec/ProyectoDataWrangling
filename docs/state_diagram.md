# State Diagrams (Mermaid)

## Dataset lifecycle

```mermaid
stateDiagram-v2
    [*] --> RAW
    RAW --> VALIDATED
    VALIDATED --> STORED
    STORED --> CLEANING
    CLEANING --> PROFILED
    PROFILED --> TRANSFORMED
    TRANSFORMED --> UNIFIED
    UNIFIED --> READY
    READY --> [*]
    anyState: ERROR
    RAW --> ERROR
    VALIDATED --> ERROR
    CLEANING --> ERROR
```
